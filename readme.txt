Калькулятор электроэнергии
Версия 4.0.0

Как установить:

1. PWA (рекомендуется)
   - Откройте calc.gotweb.ru в любом браузере (Chrome/Edge ...)
   - Нажмите на значок «Установить» в адресной строке
   - Приложение появится на рабочем столе или в меню Пуск

2. ZIP-архив
   - Скачайте calc.zip
   - Распакуйте в любую папку
   - Откройте index.html в браузере

3. Установщик Windows (EXE)
   - Скачайте установщик
   - Запустите и следуйте инструкциям
   - Приложение установится как обычная программа

Активация полной версии:
- В настройках введите ключ: electric_system
