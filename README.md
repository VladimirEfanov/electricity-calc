# Калькулятор электроэнергии ⚡

Веб-приложение для расчета стоимости электроэнергии по многотарифным счетчикам с сохранением истории, синхронизацией в облако и печатью отчетов.

## 📱 Установка на телефон

### Для iPhone:
1. Откройте сайт в **Safari**
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрутите вниз, найдите **"На экран «Домой»"**
4. Нажмите "Добавить" - иконка появится на рабочем столе

### Для Android:
1. Откройте сайт в **Chrome**
2. Нажмите меню (три точки вверху)
3. Выберите "Добавить на главный экран"
4. Подтвердите добавление

## 🌐 Онлайн доступ
**Ссылка:** [https://VladimirEfanov.github.io/electricity-calc/](https://VladimirEfanov.github.io/electricity-calc/)

## 📋 Основные возможности
- **2-тарифный и 3-тарифный учет**
- **Автоматический расчет**
- **История расчетов** с возможностью редактирования
- **Статистика** - сводные отчеты за год
- **Печать отчетов** в удобном формате
- **Режим блокировки месяцев** для последовательного учета

## 🚀 Быстрый старт
1. **Выберите тип учета** (2-тарифный или 3-тарифный)
2. **Введите показания** счетчиков
3. **Нажмите "Рассчитать"** для автоматического расчета
4. **Сохраните результат** в историю

## 📊 Как пользоваться

### Добавление новых показаний
1. **Автозаполнение** - при выборе месяца начальные показания заполняются из предыдущего месяца
2. **Ввод данных** - укажите только конечные показания (начальные уже есть)
3. **Проверка** - приложение не даст ввести конечные показания меньше начальных
4. **Сохранение** - нажмите "Сохранить" для добавления в историю

### Работа с историей
- **📝 Клик на запись** - загрузить данные в форму для редактирования
- **🗑️ Кнопка "Удалить"** - удалить одну запись
- **🧹 "Очистить всю историю"** - удалить все записи (создается полный бэкап)

### Статистика и отчеты
- Перейдите на вкладку **"Статистика"**
- Выберите год для отчетов
- Просмотрите сводку по обоим типам учета
- Нажмите "Печать отчета" для получения документа

## 💾 Импорт и экспорт данных

### 📤 Экспорт (резервное копирование)
**Зачем нужно:** Создание резервной копии всех данных для восстановления при смене устройства или очистке браузера.

**⚠️ Важно!** Перед очисткой истории автоматически создается полная резервная копия

**Как сделать:**
1. Откройте меню приложения (кнопка ≡)
2. Выберите "Экспорт данных"
3. Файл сохранится автоматически в формате JSON
4. Сохраните файл в надежном месте

### 📥 Импорт (восстановление)
**Шаги:**
1. Меню приложения → "Импорт данных"
2. Выберите файл резервной копии (.json)
3. Подтвердите импорт

## ☁️ GitHub синхронизация
**⚠️ Важно!** При синхронизации с GitHub данные полностью заменяются. Для создания резервных копий используйте экспорт в JSON.

### 🎯 Преимущества
- Резервное копирование в облаке
- Доступ с разных устройств
- История изменений

### 📝 Пошаговая настройка
1. **Создание токена на GitHub:**
   - Зайдите на GitHub → Settings → Developer settings
   - Personal access tokens → Generate new token (classic)
   - Права: Отметьте **только gist**

2. **В приложении:**
   - Нажмите на значок облака → "Подключить синхронизацию"
   - Вставьте токен → "Проверить подключение" → "Сохранить"

### 🔄 Работа с синхронизацией
**Значки статуса:**
- ✅ **Синхронизировано** - все данные актуальны
- ⚠️ **Требует внимания** - есть расхождения
- ❌ **Ошибка** - проблема с подключением
- 🔒 **Отключено** - синхронизация выключена

## 🗂️ Google Drive синхронизация (новая возможность)

Помимо GitHub, вы можете синхронизировать данные с **Google Drive** – это удобно для резервного копирования и работы с нескольких устройств, если у вас уже есть Google-аккаунт.

### 🔐 Настройка Google Drive
1. Нажмите **облако** в правом верхнем углу приложения
2. Выберите **"Войти через Google"**
3. Разрешите доступ к Google Drive (требуется для сохранения файла)
4. После авторизации данные будут автоматически синхронизироваться при каждом сохранении

### 📦 Что синхронизируется
- Вся история расчётов (обе тарифные системы)
- Настройки приложения
- Последние показания

Синхронизация работает через **папку приложения** в вашем Google Drive. Вы всегда можете найти файл резервной копии там.

## ⚙️ Режим блокировки месяцев

### 🎯 Что это?
Блокирует выбор месяцев до начала учета, предотвращая ошибки ввода данных за прошлые периоды.

**Пример:** Если учет начат с января 2026, нельзя выбрать декабрь 2025 или ранее.

**💡 Подсказка:** Чтобы добавить архивные данные - временно отключите режим в настройках, добавьте данные, затем снова включите.

## 🔧 Техническая информация

### Хранение данных
- **Локально**: Все данные хранятся в браузере (`localStorage`)
- **В облаке**: При включенной синхронизации - в приватном GitHub Gist или Google Drive
- **Резервные копии**: Автоматически создаются при опасных операциях

### Безопасность
- GitHub токен хранится только в вашем браузере
- Все gist создаются как **приватные**
- Google Drive доступ запрашивается только для создания файла приложения
- Токены имеют минимально необходимые права

### Совместимость
- **Браузеры**: Chrome, Firefox, Safari, Edge, Opera
- **Платформы**: Windows, macOS, Linux, iOS, Android
- **Требования**: Современный браузер с поддержкой JavaScript ES6+

## ❓ Частые вопросы

### Вопрос: Данные пропали после очистки кэша браузера
**Ответ**: Включите синхронизацию с GitHub или Google Drive - данные восстановятся из облака. Или используйте экспорт данных перед очисткой.

### Вопрос: Как перенести данные на другой компьютер/телефон?
**Ответ**: 
1. На старом устройстве: Меню → "Экспорт данных"
2. Сохраните файл .json
3. На новом устройстве: Меню → "Импорт данных"
4. Выберите сохраненный файл

### Вопрос: GitHub токен перестал работать
**Ответ**: Токены действительны 30 дней. Создайте новый:
1. GitHub → Settings → Developer settings → Personal access tokens
2. Отзовите старый токен
3. Создайте новый с теми же правами
4. Обновите в настройках приложения

### Вопрос: Как получить полную версию?
**Ответ**: Перейдите в Telegram-бот [@ElectricityCalcBot](https://t.me/ElectricityCalcBot), нажмите "Купить", оплатите 299 ₽ любым удобным способом. После оплаты вы получите ключ активации.
