# Калькулятор электроэнергии ⚡

Веб-приложение для расчета стоимости электроэнергии по многотарифным счетчикам с сохранением истории, синхронизацией в облако и печатью отчетов.

## 📱 Установка на телефон

### Для iPhone:
1. Откройте сайт в **Safari**
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой вверх)
3. Прокрутите вниз, найдите **"На экран «Домой»"**
4. Нажмите "Добавить" - иконка появится на рабочем столе

### Для Android:
1. Откройте сайт в **Chrome**
2. Нажмите меню (три точки вверху)
3. Выберите "Добавить на главный экран"
4. Подтвердите добавление

## 📋 Основные возможности

### ✅ 2-тарифный учет (день/ночь)
- Расчет по тарифам Т1 (день 7:00-23:00) и Т2 (ночь 23:00-7:00)
- Автоматическое заполнение начальных показаний из предыдущего месяца
- История всех расчетов с возможностью редактирования
- Печать детальных отчетов

### ✅ 3-тарифный учет (пик/полупик/ночь)
- Расчет по тарифам Т1 (пик 7:00-10:00, 17:00-21:00), Т2 (полупик 10:00-17:00, 21:00-23:00), Т3 (ночь 23:00-7:00)
- Подробная статистика потребления по зонам
- Сравнение экономии между разными тарифами

### ✅ Общие функции
- 📊 **Статистика** - сводные отчеты за год, графики потребления
- 💾 **Автосохранение** - все данные хранятся в браузере (localStorage)
- ☁️ **Синхронизация** - резервное копирование в GitHub Gist
- 🖨️ **Печать** - отчеты в удобном формате для оплаты
- 🌙 **Темная тема** - автоматическое переключение день/ночь
- 📱 **Адаптивный дизайн** - работает на любом устройстве
- 📤 **Экспорт/импорт** - перенос данных между устройствами
- 🔄 **Автообновление** - проверка новых версий из облака

## 🚀 Быстрый старт

1. **Откройте приложение** в браузере
2. **Выберите вкладку** - 2-тарифный или 3-тарифный учет
3. **Установите тарифы** - нажмите "Сменить тарифы" для ввода актуальных цен
4. **Выберите месяц и год** для расчета
5. **Введите показания** счетчиков на начало и конец месяца
6. **Нажмите "Рассчитать"** - увидите стоимость
7. **Сохраните результат** - данные сохранятся в историю

## ☁️ Настройка синхронизации (рекомендуется)

### Шаг 1: Получить GitHub токен
1. Зайдите на [GitHub.com](https://github.com) → Settings → Developer settings
2. Выберите "Personal access tokens" → "Tokens (classic)"
3. Нажмите "Generate new token"
4. Введите название: "Electricity Calculator"
5. Выберите **только один scope: `gist`**
6. Скопируйте токен (выглядит как `ghp_xxxxxxxxxxxxxxxxxxxx`)

### Шаг 2: Ввести токен в приложении
1. Нажмите на иконку **облака** в правом верхнем углу
2. Выберите "Подключить синхронизацию"
3. Вставьте ваш GitHub токен
4. Нажмите "Проверить подключение"
5. Если успешно - нажмите "Сохранить"

✅ **Готово!** Теперь ваши данные автоматически сохраняются в приватном облаке GitHub.

## 📊 Как пользоваться

### Добавление новых показаний
1. **Автозаполнение** - при выборе месяца начальные показания заполняются из предыдущего месяца
2. **Ввод данных** - укажите только конечные показания (начальные уже есть)
3. **Проверка** - приложение не даст ввести конечные показания меньше начальных
4. **Сохранение** - нажмите "Сохранить" для добавления в историю

### Работа с историей
- **📝 Клик на запись** - загрузить данные в форму для редактирования
- **🗑️ Кнопка "Удалить"** - удалить одну запись
- **🖨️ "Печать истории"** - распечатать все записи в табличном виде
- **🧹 "Очистить всю историю"** - удалить все записи (создается полный бэкап)

### Статистика и отчеты
- Перейдите на вкладку **"Статистика"**
- Выберите год для отчетов
- Просмотрите сводку по обоим типам учета
- Нажмите "Печать отчета" для получения документа

## 🔧 Техническая информация

### Хранение данных
- **Локально**: Все данные хранятся в браузере (`localStorage`)
- **В облаке**: При включенной синхронизации - в приватном GitHub Gist
- **Резервные копии**: Автоматически создаются при опасных операциях

### Безопасность
- GitHub токен хранится только в вашем браузере
- Все gist создаются как **приватные**
- Никто кроме вас не имеет доступа к вашим данным
- Токен имеет права **только на работу с gist**

### Совместимость
- **Браузеры**: Chrome, Firefox, Safari, Edge, Opera
- **Платформы**: Windows, macOS, Linux, iOS, Android
- **Требования**: Современный браузер с поддержкой JavaScript ES6+

## ❓ Частые вопросы

### Вопрос: Данные пропали после очистки кэша браузера
**Ответ**: Включите синхронизацию с GitHub - данные восстановятся из облака. Или используйте экспорт данных перед очисткой.

### Вопрос: Как перенести данные на другой компьютер/телефон?
**Ответ**: 
1. На старом устройстве: Меню → "Экспорт данных"
2. Сохраните файл .json
3. На новом устройстве: Меню → "Импорт данных"
4. Выберите сохраненный файл

### Вопрос: GitHub токен перестал работать
**Ответ**: Токены действительны 30 дней. Создайте новый:
1. GitHub → Settings → Developer settings → Personal access tokens
2. Отзовите старый токен
3. Создайте новый с теми же правами
4. Обновите в настройках приложения

### Вопрос: Приложение не работает на старом телефоне
**Ответ**: Убедитесь, что:
1. Браузер обновлен до последней версии
2. Включен JavaScript
3. Попробуйте другой браузер (Chrome вместо Safari)
